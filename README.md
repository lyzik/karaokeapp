# Karaoke App
## About

This website is used for the popular karaoke. Using spotify api and lyrics finder npm package, you can sing your favorites songs!

## Techologies:
 - React
 - Next.js
 - TypeScript
 - Styled JSX
 - Spotify API
 - lyrics finder
 - NextAuth
